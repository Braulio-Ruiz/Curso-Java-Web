Lección 16 30 min

Interfaces Funcionales y Lambdas

Escribe menos, haz más. El cambio de paradigma de imperativo a declarativo.

Objetivos de Élite

Clases Anónimas vs Lambdas.
Interfaces: Predicate, Consumer, Function.
Method References (::).
Código como datos.

1. Teoría Suprema: Código de Primera Clase

Hasta Java 7, para pasar una acción a un botón, tenías que crear un objeto entero. Las Lambdas permiten tratar bloques de código como si fueran objetos, usando la instrucción `invokedynamic` para evitar el costo de memoria de las clases anónimas.

1.1 El Kit de Herramientas Funcional

Predicate<T>

Pregunta. Devuelve boolean.
x -> x > 10

Function<T, R>

Transforma. T a R.
s -> s.length()

Consumer<T>

Ejecuta. Efecto secundario.
msg -> print(msg)

Supplier<T>

Provee. Lazy loading.
() -> Math.random()

2. Laboratorio de Código

LambdasPro.java 
public class LambdasPro {
    public static void main(String[] args) {
        // Estilo Antiguo (Verboso)
        Runnable r1 = new Runnable() {
            public void run() { System.out.println("Viejo"); }
        };

        // Estilo Moderno (Lambda) - Inferencia de tipos
        Runnable r2 = () -> System.out.println("Nuevo");

        // Method Reference (::)
        List nombres = Arrays.asList("Ana", "Bea");
        nombres.sort(String::compareTo); // En vez de (a,b) -> a.compareTo(b)
        nombres.forEach(System.out::println);
    }
}

Cadena de Montaje (Composición)

Crea una función que duplique y otra que sume 10. Únelas con `.andThen()`.

Function duplicar = x -> x * 2;
Function sumarDiez = x -> x + 10;

// f(g(x)) -> Primero duplica, luego suma
Function tuberia = duplicar.andThen(sumarDiez);
System.out.println(tuberia.apply(5)); // 20
Lección 17

Stream API

La cinta transportadora de datos. Procesamiento declarativo y paralelo.

1. Teoría Suprema: La Cinta Transportadora

Imagina una fábrica. Vuelcas las piezas (`List`) en una cinta (`.stream()`). Los robots (`filter`, `map`) procesan las piezas en el camino hacia la caja final (`collect`).

Evaluación Perezosa (Lazy)

Si no pones la caja al final (Operación Terminal), la cinta no arranca. Esto permite optimizaciones masivas. 

List resultado = empleados.stream()
    .filter(n -> n.startsWith("A"))      // Robot 1: Filtra
    .map(String::toUpperCase)            // Robot 2: Pinta (Transforma)
    .sorted()                            // Robot 3: Ordena
    .collect(Collectors.toList());       // Caja Final (Terminal)
Lección 18

Manejo de Nulidad (Optional)

1. Teoría Suprema: La Caja Cuántica

`Optional<T>` es un contenedor que puede o no tener un valor. Te obliga a pensar en el caso "vacío" en tiempo de compilación.

Regla de Oro

NUNCA uses get() sin verificar isPresent(). Usa orElse() o ifPresent(). 

Optional resultado = buscarUsuario(99);

// Estilo Funcional (Seguro)
resultado.ifPresent(u -> System.out.println("Encontrado: " + u));

// Valor por defecto
String nombre = resultado.orElse("Invitado");
Lección 19

Date & Time API

1. Teoría Suprema: El Tiempo es Relativo

La API moderna (`java.time`) es inmutable y segura. Distingue entre tiempo máquina y tiempo humano.

Instant (Máquina)

Nanosegundos desde 1970 (UTC). Universal para logs.

LocalDate (Humano)

"25 de Diciembre". Depende de la zona horaria.

// Inmutabilidad: Las operaciones crean nuevas copias
LocalDate hoy = LocalDate.now();
LocalDate vencimiento = hoy.plusDays(30);

// Zonas Horarias
ZonedDateTime tokio = Instant.now().atZone(ZoneId.of("Asia/Tokyo"));
